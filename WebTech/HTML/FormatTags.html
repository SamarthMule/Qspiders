Here are some of the most common HTML tags:

Bold Tag : Make the text bold, An Inline element.
Example : Bold text
Strong Tag : Make the text strong, An Inline element. Same as bold tag but tells that particular content is important to user and browser
Example : Important text

Italic Tag : Make the text italic, An Inline element. Does not provide semantic meaning
Example : Italic text
Emphasized Tag : Make the text emphasized, An Inline element. Provides semantic meaning
Example : Emphasized text
